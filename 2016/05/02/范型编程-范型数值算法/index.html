范型数值算法

头文件

包含

  • accumulate
  • inner_product
  • partial_sum
  • adjacent_difference

accumulate(1)

  • 对区域内的每个元素进行累加
  • 设result=_Val,result += *it

accumulate(2)

  • 每个元素累加
  • result=_Val, result = Func(*it, result);

inner_product(1)

  • 两个区域
  • result = _Val , result = result + (it) × (_First2 + (it1 - _First1 ) );

inner_product(2)

  • result初始值为_Val
  • result = _Func1( result, Func2( it, (_First2 + ( it1 - _First1 ) ) ) )

partial_sum

  • 除了第一个对象外, 当前对象等于当前对象加上 上一个 对象 的结果.
  • 设it = _First, it = _First, (it+1)= _First + *( _First + 1 )…依次类推

partial_sum(2)

  • 把(1)中的 加法 换成自定义函数 _Func

adjacent_difference(1)

  • 除了第一个对象外,当前对象=当前对象 减去 上一个对象

adjacent_difference(2)

  • 把(1)中的 减法 换成自定义函数 _Func
